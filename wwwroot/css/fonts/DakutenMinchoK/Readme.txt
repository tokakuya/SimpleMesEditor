【濁点明朝について】
濁点がつかない、「あいえお」といったひらがなやカタカナに濁点をつけた文字を収録しています。
通常濁点のつかないひらがなカタカナを濁点のついたひらがなカタカナに置き換えています。

IPAex明朝で入力した文字のフォントを一部濁点明朝Kに置き換えることで濁点付き文字を手軽に入力できます。
フォントの切り替えかたはお使いのソフトウェアのマニュアルを参考に修正してください。

【IPAフォントへの戻しかた】
1.濁点明朝Kをアンインストールします
2.http://ipafont.ipa.go.jp/node17#jp よりIPAex明朝をダウンロードします
3.IPAex明朝をインストールします

【派生プログラムを作成する過程でフォント開発プログラムによって作成された追加のファイルであって派生プログラムをさらに加工するにあたって利用できるファイル】
http://twiren.sakura.ne.jp/font/ より、Font Forgeによって生成された追加のファイルDakutenMinchoK.sfdをダウンロードできます。
濁点明朝をさらに加工する際に、このファイルを利用できます。
